Провести ABCXYZ-анализ ассортимента пиццерии за 2015 год для определения наиболее и наименее значимых товаров с целью оптимизации ассортимента.
1.1 Основной отчет ABC-анализа:
Наименование пиццы  — pizza_name.
Общее количество проданных единиц за 2015 год — full_quantity.
Общая выручка за 2015 год — full_revenue.
Категория ABC по количеству проданных единиц — abc_category_by_units.
Категория ABC по выручке — abc_category_by_revenue.

1.2 Сводная статистика по комбинированным категориям:
Комбинированная категория в формате "AA", "BC", где:
Первая буква — категория по количеству.
Вторая буква — категория по выручке.
Количество пицц в каждой комбинации  — cnt_of_pizzas.
Общий объем продаж в каждой комбинации  — total_quantity.
Общая выручка в каждой комбинации — total_revenue.

Примечания:
Пороговые значения 0.8 и 0.95 могут быть скорректированы в зависимости от задачи.
Для более детального анализа можно дополнительно сегментировать данные по размерам пицц.
Полученные результаты можно рассматривать в комплексе с XYZ-анализом для планирования запасов.

2.1 XYZ-анализ — метод классификации товаров по стабильности/предсказуемости спроса
Коэффициент вариации (CV) — относительный показатель разброса данных, показывающий, насколько сильно значения отклоняются от среднего значения
Рассчитывается как: стандартное отклонение / среднее значение
Чем меньше CV → тем стабильнее продажи
Чем больше CV → тем нестабильнее продажи
Стандартное отклонение — абсолютный показатель, характеризующий разброс данных вокруг среднего значения

2.2 Пример расчета коэффициента вариации:
Пицца "A": продажи по месяцам: [100, 105, 98, 102, 101] → среднее = 101.2, ст.отклонение = 2.6 → CV = 2.6/101 ≈ 0.026 (2.6%) → очень стабильные продажи
Пицца "B": продажи по месяцам: [20, 150, 5, 80, 45] → среднее = 60, ст.отклонение = 58 → CV = 58/60 ≈ 0.97 (97%) → очень нестабильные продажи
